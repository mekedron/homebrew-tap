# Homebrew Tap

Personal Homebrew tap with CLI tools and apps.

## Setup

```bash
brew tap mekedron/tap
```

## Formulas

| Formula | Version | Description | Platforms |
|---------|---------|-------------|-----------|
| [`wolt-cli`](https://github.com/mekedron/wolt-cli) | 2.1.2 | Unofficial community CLI for interacting with Wolt APIs | macOS, Linux |
| [`otta-cli`](https://github.com/mekedron/otta-cli) | 0.2.3 | CLI for automating workflows around otta.fi time tracking | macOS, Linux |
| [`ocr`](https://github.com/mekedron/ocr) | 1.0.0 | Capture a screenshot, OCR it with Apple Vision, and print text to stdout | macOS |

```bash
brew install mekedron/tap/wolt-cli
brew install mekedron/tap/otta-cli
brew install mekedron/tap/ocr
```

## Casks

| Cask | Version | Description | Requires |
|------|---------|-------------|----------|
| [`ClipSlop`](https://github.com/mekedron/ClipSlop) | 2.0.0 | Keyboard-first AI text pipeline for macOS | macOS Sonoma+ |
| [`HiddenBarIcons`](https://github.com/mekedron/HiddenBarIcons) | 1.0.0 | Reveal menu bar icons hidden under the MacBook notch | macOS Sonoma+ |
| [`PieSwitcher`](https://github.com/mekedron/PieSwitcher) | 1.1.0 | Radial pie-menu window switcher for macOS | macOS Sonoma+ |

```bash
brew install --cask mekedron/tap/clipslop
brew install --cask mekedron/tap/hiddenbaricons
brew install --cask mekedron/tap/pieswitcher
```
